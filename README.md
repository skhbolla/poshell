# poshell
POSIX style shell built to explore the Linux Process API. This project implements the core mechanics of a command-line interface, including process creation via fork/exec, pipeline management with pipe, history persistance and autocompletion features. Designed as a deep-dive into how the kernel manages user-space execution environments.

Refer [IEEE 1003.1 standard](https://pubs.opengroup.org/onlinepubs/9699919799/utilities/V3_chap02.html)
